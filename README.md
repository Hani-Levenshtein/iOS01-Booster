## 설치 방법
   **Booster를 직접 사용해 보아요 🚀🔥**
1. Xcode 13(권장) 이상과 Cocoapods이 설치되어있어야 합니다.
    > `brew install cocoapods` - Cocoapods 설치
2. 프로젝트를 로컬에 클론 하게 되면 프로젝트 디렉토리에서 
    `pod install`을 실행 시켜 주세요.
3. 이후 Booster.xcworkspace를 실행 한 이후 `command + R`을 통해 체험 가능합니다.
